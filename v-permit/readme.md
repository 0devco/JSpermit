# v-permit directive Vue (JSpermit)


# install

```bash
npm -i @0devco/Vpermit
# yarn
yarn add @0devco/Vpermit
```

# import > App.js

```js
import Vpermit from '@0devco/Vpermit';
Vue.use(Vpermit);
```

# use `v-permit`

```html
<div v-permit="true/false">
  <h1> Done, you can access this </h1>
</div>
```
